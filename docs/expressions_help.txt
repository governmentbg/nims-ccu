Параметри 
  Числови - за бюджетен ред от 3 Ниво
    [БФП]
    [СФ]
    [Общо]

  Текстови - за бюджетен ред от 3 Ниво
    [01] -> код 1 измерение на реда
    [02] -> код 2 измерение на реда
    [03] -> код 3 измерение на реда
    [04] -> код 4 измерение на реда
    [05] -> код 5 измерение на реда
    [06] -> код 6 измерение на реда
    [07] -> код 7 измерение на реда
    [NUTS2] -> код NUTS2 8 измерение  на реда
    [Приоритетна ос] -> код приоритетна ос  на реда
    [Финансов източник] -> код финасов източник  на реда
    [Режим на помощта]  -> наименование Режим на помощта  на реда
    
  Текстови - навсякъде
    [Категория/статус на предприятието] -> Наименование на Категория/статус на предприятието на Кандидата
    [Номер] -> Номер на Кандидата
    [Тип на организацията] -> Наименование на Тип на организацията на Кандидата
    [Вид организация] -> Наименование на Вид организация на Кандидата
    [Код на организацията по КИД 2008] -> Код на организацията по КИД 2008 на Кандидата
    [Код на проекта по КИД 2008] -> Код на проекта по КИД 2008 на Кандидата
    
    Пример
      [01] == '002'
    
  Булеви - за бюджетен ред от 3 Ниво
    [Допустим разход] -> стойност Допустим разход на реда
  Булеви - ПП
    [Частно правна]  -> стойност Частно правна на Кандидата
    
    Пример
      [Допустим разход] == [Да]

Константи 
  Булеви - навсякъде
    [Да] - true
    [Не] - false 
    [Винаги] - true
    [Всички] - true

Функции
  "Min"	Returns the smaller of two numbers.	Min(1, 2)	1
  "Max"	Returns the larger of two specified numbers.	Max(1, 2)	2
  "Pow"	Returns a specified number raised to the specified power.	Pow(3, 2)	9d
  "Abs"	Returns the absolute value of a specified number.	Abs(-1)	1M
  "Truncate"	Calculates the integral part of a number.	Truncate(1.7)	1
  "In"	Returns whether an element is in a set of values.	in(1 + 1, 1, 2, 3)	true
  "If"	Returns a value based on a condition.	if(3 % 2 = 1, 'value is true', 'value is false')	'value is true'
  
  sum(node, selector, condition)
  sumGrand(node)
  sumSelf(node)
  sumTotal(node)
    Сумира посочената стойност от всички редове на 3-то ниво, отговарящи на условието, към клона на бюджета.  
    node:           идентификатор на ред. Пример : [Бюджет], [VI]..., [2]..., [Група]*
    selector:       каква стойност се взема от реда. Пример : [БФП], [СФ], [Общо]
    condition:      условие за редовете от 3-то ниво. Пример : [Допустим разход] == [Да]
    Пример : sum([VI], [БФП], [Допустим разход] == [Да]) - Сумата на БФП на всички бюджетни редове от 3-ти ред към VI, които са допустим разход.

  any/all(nodeOrLevel, condition)
    Когато работи за групи
    Проверява дали "поне една"/"всички" от групи отговарящи на условието. 
    nodeOrLevel:    идентификатор на група. Пример : [Ниво 1],[Ниво 2]
    selector:       каква стойност взима за всяка група. Пример : sum([Група], [БФП], [Всички])
    condition:      филтър за групите, изпълнява се за всяка група. Пример : sum([Група], [БФП], [Всички])
    Пример :  any([Ниво 1], sum([Група], [БФП], [Всички]) > 1000) - Поне едно римско да има БФП повече от 1000
              all([Ниво 2], sum([група], [Общо], [Всички]) < 1000) - Всички арабски да имат обща стойност по-малко от 1000

    Когато работи за 3-то ниво
    Проверява дали "поне един"/"всички" от редове на 3-то ниво отговарят на условието.
    nodeOrLevel:    идентификатор на ред. Пример : [Бюджет], [VI]..., [2]...
    condition:      условие за редовете от 3-то ниво. Пример : [Допустим разход] == [Да]
    Пример :  all([Бюджет], [БФП] > 0) - Всички бюджетни редове от 3-ти ред да имат БФП повече от 0

Примери :
Съобщение : По процедурата е недопустимо категорията на предприятието да бъде "Голямо" или "Неприложимо"! Предприятието може да бъде "Микро", "Малко" или "Средно".
Условие   : 
Проверка  : in([Категория/статус на предприятието], 'Малко', 'Микро', 'Средно')
--
Съобщение : Минималният размер на безвъзмездната финансова помощ за микро предприятие е 100 000 лв.	
Условие   : ([Категория/статус на предприятието]) = 'Микро'	
Проверка  : (sum([Бюджет], [БФП], ([Допустим разход]) = ([ДА]))) >= 100000
--
Съобщение : Максималният интензитет на безвъзмездната финансова помощ за микро и малко предприятие извън Югозападен район е в размер на 70% .	
Условие   : in([Категория/статус на предприятието], 'Малко', 'Микро')
Проверка  : (sum([Бюджет], [БФП], (([Допустим разход]) = ([ДА])) and (([NUTS2]) != 'BG41'))) <= ((sum([Бюджет], [ОБЩО], (([Допустим разход]) = ([ДА])) and (([NUTS2]) != 'BG41'))) * 0.70)
--
Съобщение : За недопустимите разходи стойността на БФП трябва да е равна нула.	
Условие   : 
Проверка  : all([Бюджет], ((([БФП]) = 0) and (([Допустим разход]) = ([Не]))) or (([Допустим разход]) = ([Да])))
--
Съобщение : Разходи за консултантски услуги по изготвяне на проектното предложение по настоящата процедура заложени в точка 3 не могат да бъдат повече от 10 000 лева.
Условие   :
Проверка  : (sumTotal([3])) <= 10000
--
Съобщение : Разходи за одит на проекта e допустим разход и се въвежда в точка 4 само за проекти с размер на безвъзмездната финансова помощ над 750 000 евро/1 466 872,50 лв.	
Условие   : (sumTotal([Бюджет])) < 1466872.50	
Проверка  : (sumTotal([4])) = 0
--
Съобщение : Разходи за организация и управление заложени в точка 6 трябва да са под 5% от преките допустими разходи по точки 1 и 2		
Условие   :
Проверка  : (sumTotal([6])) <= (((sum([1], [Общо], ([Допустим разход]) = ([Да]))) + (sum([2], [Общо], ([Допустим разход]) = ([Да])))) * 0.05)
